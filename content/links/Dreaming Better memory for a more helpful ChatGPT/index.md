description: "- OpenAI is deploying an upgraded \"Dreaming\" memory architecture that automatically synthesizes and updates user context across conversations, eliminating reliance on explicit recall triggers.
- The system is evaluated against three core metrics: accurately retaining factual details, consistently applying stated or implicit preferences, and dynamically adjusting memories as time passes.
- Compute optimizations reduced serving requirements by roughly 5x, enabling a scalable rollout to Free and Go users while expanding memory capacity for Plus and Pro subscribers.
- An updated interface allows users to review, edit, and manage synthesized memories directly, with the option to drill into specifics through conversation.

This release transitions ChatGPT from manual note-taking to a background-driven system that continuously curates context without requiring explicit prompts. By prioritizing temporal accuracy and preference alignment, OpenAI addresses long-standing issues of memory staleness while making multi-year interactions more coherent and personalized at scale. The architecture now functions as a unified foundation across all subscription tiers, reflecting both technical maturation and the company’s commitment to delivering reliable, widely accessible AI assistance."
